Materi CSS

1. Sejarah dan Pengertian CSS

Pengertian CSS: CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mendesain tampilan dan format halaman web yang ditulis dalam HTML. Dengan CSS, kita bisa mengatur warna, font, tata letak, dan gaya visual lainnya.

Sejarah CSS: CSS pertama kali dikembangkan pada tahun 1996 oleh World Wide Web Consortium (W3C). Versi pertama (CSS1) hanya mendukung format dasar. CSS2 (1998) dan CSS3 (2001) memperkenalkan fitur yang lebih kompleks seperti tata letak grid, efek animasi, dan lainnya.

2. Dasar-Dasar CSS

  • Struktur CSS:
    • Selektor (Selector): Menentukan elemen HTML mana yang akan di-styling.
Deklarasi (Declaration): Terdiri dari properti (property) dan nilai (value).
Aturan CSS (CSS Rule): Kombinasi selektor dan deklarasi.

/* Contoh aturan CSS */
p {
    color: blue;
    font-size: 16px;
}
  • Cara Mengintegrasikan CSS dengan HTML:
    • CSS Inline: Menulis CSS langsung di elemen HTML.
      • <p style="color: red;">Teks berwarna merah.</p>
    • CSS Internal: Menulis CSS di dalam tag <style> pada bagian <head>.
      • <style>
    body {
        background-color: lightblue;
    }
</style>
    • CSS Eksternal: Menyimpan CSS dalam file .css terpisah dan menghubungkannya di HTML.
      • <link rel="stylesheet" href="style.css">

3. Selektor CSS

  • Selektor Tag: Memilih semua elemen HTML dengan nama tag tertentu.
    • h1 { color: blue; }
  • Selektor ID: Memilih elemen dengan atribut id tertentu.
    • #header { background-color: grey; }
  • Selektor Class: Memilih elemen dengan atribut class tertentu.
    • .content { font-size: 14px; }
  • Selektor Atribut: Memilih elemen berdasarkan atribut tertentu.
    • input[type="text"] { color: green; }

4. Konsep Tata Letak (Layout) dalam CSS

  • Display:
    • Block dan Inline: Mengatur cara elemen ditampilkan.
      • display: block; /* Mengisi ruang secara penuh */
display: inline; /* Mengisi ruang sesuai konten */
    • Flexbox: Digunakan untuk membuat tata letak fleksibel.
      • display: flex;
justify-content: center;
align-items: center;
    • Grid: Mengatur layout berbasis grid yang dinamis.
      • display: grid;
grid-template-columns: 1fr 1fr;
grid-gap: 10px;

5. Responsivitas dengan CSS

  • Media Query: Digunakan untuk menyesuaikan tampilan berdasarkan ukuran layar.
    • @media (max-width: 600px) {
    body {
        background-color: lightblue;
    }
}
  • Unit Relatif dan Tetap:
    • Unit Tetap: px (piksel), cm (centimeter).
    • Unit Relatif: %, vw (viewport width), vh (viewport height), em, rem.

6. Fitur dan Teknik CSS Tingkat Lanjut

  • Animasi dan Transisi:
    • /* Contoh Transisi */
box {
    transition: all 0.3s ease;
}

/* Contoh Animasi */
@keyframes example {
    from { background-color: red; }
    to { background-color: yellow; }
}
.box {
    animation: example 3s infinite;
}
  • PseudoClass dan PseudoElement:
    • a:hover { color: red; } /* Mengubah warna link saat dihover */
p::first-line { font-weight: bold; } /* Mengubah gaya teks baris pertama paragraf */
  • CSS Variables:
    • :root {
    --main-bg-color: #f0f0f0;
}
body {
    background-color: var(--main-bg-color);
}
  • CSS Gradients: Membuat gradien warna untuk latar belakang elemen.
    • background: linear-gradient(to right, red, yellow);

Kesimpulan

CSS adalah bahasa stylesheet yang mengatur tampilan halaman web yang telah dibangun dengan HTML. CSS memungkinkan kita mengatur warna, font, tata letak, dan efek visual untuk menciptakan desain yang responsif, elegan, dan user-friendly.
