Materi HTML

1. Sejarah dan Pengertian HTML

Pengertian HTML: HTML (HyperText Markup Language) adalah bahasa standar untuk membuat halaman web. HTML mendefinisikan struktur dan layout halaman web dengan menggunakan elemen-elemen seperti heading, paragraf, link, gambar, dan lainnya. HTML bukanlah bahasa pemrograman, melainkan bahasa markup yang memberikan instruksi kepada browser bagaimana menampilkan konten.

Sejarah HTML:

  • Diciptakan oleh Tim Berners-Lee (1991): HTML dikembangkan pertama kali oleh Tim Berners-Lee. Pada tahun 1991, ia merilis versi pertama HTML sebagai alat untuk berbagi informasi dalam jaringan World Wide Web (WWW).
  • HTML 2.0 (1995): Versi resmi pertama yang distandarisasi oleh Internet Engineering Task Force (IETF), menambahkan fitur seperti form.
  • HTML 3.2 dan 4.0 (1997 dan 1999): Memperkenalkan elemen tata letak kompleks seperti tabel dan mendukung CSS.
  • XHTML (2000): Evolusi dari HTML 4.0 dengan prinsip-prinsip XML.
  • HTML5 (2014): Versi terbaru dan paling canggih dari HTML, memperkenalkan elemen multimedia dan struktur semantik.

2. Struktur Dasar HTML

Setiap halaman HTML memiliki struktur dasar yang sama, yang terdiri dari elemen-elemen berikut:

  • <!DOCTYPE html>: Deklarasi bahwa dokumen ini menggunakan HTML5.
  • <html>: Elemen root yang mencakup seluruh konten.
  • <head>: Bagian metadata (seperti judul, link ke stylesheet, charset).
  • <body>: Tempat konten halaman web ditampilkan.
<!DOCTYPE html>
<html>
  <head>
    <title>Judul Halaman</title>
  </head>
  <body>
    Konten halaman web.
  </body>
</html>

3. Elemen Dasar HTML

  • Heading: Digunakan untuk membuat judul dalam berbagai ukuran, dari <h1> (terbesar) hingga <h6> (terkecil).
    • <h1>Ini Heading 1</h1>
<h2>Ini Heading 2</h2>
...
<h6>Ini Heading 6</h6>
  • Paragraf: Elemen <p> digunakan untuk membuat blok teks sebagai paragraf.
    • <p>Ini adalah sebuah paragraf.</p>
  • Link: Elemen <a> digunakan untuk membuat hyperlink.
    • <a href="https://www.example.com">Ini adalah link</a>
  • Gambar: Elemen <img> digunakan untuk menampilkan gambar.
    • <img src="gambar.jpg" alt="Deskripsi gambar">
  • Daftar: <ol> untuk daftar bernomor dan <ul> untuk daftar dengan bullet.
    • <ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

4. Fitur Lanjutan HTML

  • Multimedia: Elemen <audio> dan <video> untuk menambahkan media.
    • <audio controls>
  <source src="audio.mp3" type="audio/mp3">
</audio>

<video controls>
  <source src="video.mp4" type="video/mp4">
</video>
  • Canvas: Elemen <canvas> digunakan untuk menggambar grafik menggunakan JavaScript.
    • <canvas id="myCanvas" width="200" height="100"></canvas>
<script>
  let c = document.getElementById("myCanvas");
  let ctx = c.getContext("2d");
  ctx.fillStyle = "#FF0000";
  ctx.fillRect(20, 20, 150, 100);
</script>

Kesimpulan

HTML adalah bahasa markup yang membangun struktur dasar halaman web. Elemen seperti heading, paragraf, gambar, dan link menyusun konten halaman, memberikan hierarki informasi, dan menyiapkan fondasi bagi desain visual.
